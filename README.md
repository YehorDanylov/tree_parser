# tree_parser

## Опис

Цей проєкт реалізує парсер арифметичних виразів із врахуванням пріоритетів операцій та дужок. Парсер приймає математичний вираз у вигляді рядка (наприклад, `3 + 5 * (2 - 8) / 4`) і перетворює його в абстрактне синтаксичне дерево (AST), яке потім можна обчислити, щоб отримати числовий результат.

## Що Парситься?

Парсер зчитує вирази, які можуть містити:

- Цілі числа
- Бінарні оператори: `+`, `-`, `*`, `/`
- Дужки для зміни порядку виконання операцій

## Граматика 

Expr = Term { ("+" | "-") Term } ;
Term = Factor { ("*" | "/") Factor } ;
Factor = Number | "(" Expr ")" ;
Number = digit { digit } ;
